## ビンゴカードを作ってみよう

#### 1. class Bingo
- Field
	- boolean ビンゴカードの当否
	- int ビンゴカードの数字
- Constructor  
	boolean,intの2つを渡してインスタンスを生成
#### 1. BingoCardのデザイン

|列1|列2|列3|列4|列5|
|---|---|---|---|---|
|1|16|31|46|61|
|2|17|32|47|62|
|7|22|0|53|68|
|14|29|44|59|74|
|15|30|45|60|75|

列1 1~15
列2 16~30
列3 31~45
列4 46~60
列5 61~75  
例外として中央は0
